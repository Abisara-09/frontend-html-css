CSS Transition: allow you to change property values smoothly over a given duration.

  1. Transition: shorthand property
  2. Transition-delay:it species when will the transition effect start.
  3. Transition-durationit specifies that for how much time the effect would last
  4. Transition-property: it specifies the css property to which you want to applythe transition to
  5. Transition-timing-function: it specifies the speedcurve of the transition effect
    • ease, linear, ease-in, ease-out,ease-in-out,cubic-bezier
